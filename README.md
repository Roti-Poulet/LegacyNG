# LegacyNG
A project that aims to restore compatibility of the game BeamNG with older OS like Windows XP and Windows 7

If you use LegacyNG for a YouTube video, please add the link to this repository in both your video and your video description.

For Windows XP, One-Core-API 4.x.x is NEEDED
For Windows 7, VxKex is , at least for now, NEEDED

Compatibility: 
Windows XP x64: Up to BeamNG 0.34 [0.36 is experimental with a lot of bugs and crashes]

Windows Vista: Planned to do, WIP

Windows 7 x64: Up to BeamNG 0.38 (latest)

THIS PROJECT IS CURRENTLY WORK IN PROGRESS, EXPECT MANY CRASHES AND RANDOM BUGS !!!!


How to install?
1) Download the latest release, depending on what OS you are on
2) Extract the ZIP file, copy everything in it
3) Paste all files into your BeamNG Bin64 folder
4) Launch the game by double clicking on the BeamNG executable (still in the Bin64 folder, the launcher may not work). If you try an alpha version of this patch, launch this executable with -nocefsandbox and -disable-sandbox
